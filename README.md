## Command Line Interface

#### Start Vite dev server in the current directory.

```
   npm run dev
```

---

#### Build for production.

```
   npm run build
```

---

#### Locally preview the production build. Do not use this as a production server as it's not designed for it.

```
   npm run preview
```
